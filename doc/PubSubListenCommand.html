PubSubListenCommand

Purpose

Set up a message receiver fot specified pubsub subscription that triggers execution of a script

Variants

UsageDescription
SUBSCRIBE subscription script *triggers script in current workspace
SUBSCRIBE subscription IN folder script *triggers script in sub-workspace specified by given folder

Arguments

ArgumentValue
subscriptionThe name of the subscription from which messages will be received
scriptThe path to the script (relative to current workspace, not the execution workspace) that gets executed when trigger receives a GET request
folderThe path to a folder (relative to current workspace) that serves as execution workspace (the executed script can be located outside this folder)
*Names of arguments expected in messages

Plugin

This command is defined in PubSubPlugin
